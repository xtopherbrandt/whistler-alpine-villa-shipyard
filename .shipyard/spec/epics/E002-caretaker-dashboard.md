# Caretaker Dashboard

## Description

A mobile-first dashboard giving the caretaker real-time visibility into occupancy across all 52 units in 7 buildings and all vehicles registered in the parking lot. The caretaker uses this in the field to verify that units with activity have a corresponding check-in record and that all vehicles in the lot are registered to a current stay.

Refresh-on-demand model (no push updates required).

## Business Value

Replaces the caretaker's reliance on the paper notebook and verbal check-ins. Gives them a single screen they can check from anywhere on the property to verify the expected vs actual state of the complex.

## Dependencies

Requires E001 (Occupancy & Check-In) — occupancy data must exist to display.
Requires E005 (User & Access Management) — caretaker account must exist.
